# 🚀 E-commerce Gaming

E-commerce dedicato alla vendita di attrezzature da gaming, pensato per un’esperienza simile a quella utilizzata dai pro player.
</br>
Il sistema include autenticazione utenti, gestione profilo, storico ordini e fatturazione, replicando il flusso completo di un e-commerce moderno.

---

## 🧠 - Overview

- E-commerce per attrezzature da gaming
- Esperienza utente completa con gestione account e acquisti
- Sistema di ordini, fatturazione e profili utente
- Struttura ispirata a un e-commerce reale

## ⚙️ Tech Stack

### 🌐 Frontend
- React
- Redux
- Sass / Bootstrap

### 🔧 Backend
- Java
- Spring Boot
- Spring Security
- JWT Authentication

### 🗄️ Database
- PostgreSQL

### 🧪 Tools
- Postman
- Node.js
- Git

## 🔐 Features

- Sistema di autenticazione utenti (login / register)
- Gestione profilo utente
- Catalogo prodotti con visualizzazione dettagli
- Carrello e gestione ordini
- Storico ordini per utente
- Sistema di fatturazione
- API REST sicure con Spring Security
- Autenticazione basata su JWT
- Interfaccia responsive e moderna

## ⚙️ Setup

### 📦 1. Clona il progetto
```bash
git clone https://github.com/luke-wb/Capstone-Project.git
```

### 🗄️ 2. Crea il database PostgreSQL
```bash 
CREATE DATABASE CapstoneProject;
```

### 🔧 3. Avvia il backend
```bash 
cd Capstone-BE
mvn clean install
mvn spring-boot:run
```

### 🌐 4. Installa il frontend
```bash 
cd Capstone-FE
npm install
```

### 🚀 5. Avvia il frontend
```bash 
npm start
```
